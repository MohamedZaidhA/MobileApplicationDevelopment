# Experiment 1 - Mobile Application Development

This folder contains the files and resources for Experiment 1 of the Mobile Application Development.

## Contents

This experiment focuses on foundational mobile app development concepts.
This app contains Three Buttuons which acts three different functions,
1. Changes Text Size.
2. Changes Text Color.
3. Changes Background Color.

## Set Up

Just download the folder, Open it in Android Studio run and experience the App
