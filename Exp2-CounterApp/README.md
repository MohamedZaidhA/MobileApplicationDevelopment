# Experiment 2 - Mobile Application Development

This folder contains the files and resources for Experiment 2 of the Mobile Application Development.

## Contents

This experiment focuses on foundational mobile app development concepts.
This app contains Three Buttons which acts three different functions,
1. Increments the count.
2. Decrements the count.
3. Resets the count.

## Set Up

Just download the folder, Open it in Android Studio run and experience the App
